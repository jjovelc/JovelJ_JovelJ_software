# Getting Geneious Enterprise up and running

**1. Create Geneious account**

Go to [Genious](https://www.geneious.com) to create an user account.
    - Click on `Sign in`
    - Click on `Geneious Prime My Account`

<div align="center">
   <img src="images/geneious-1.png" width="800" alt="Quality Scores Line Plot">
</div>

Create an account if you don't already have one (just following the simple instructions provided).

<div align="center">
    <img src="images/geneious-2.png" width="800" alt="Quality Scores Line Plot">
</div> 

If you successfully created an account you should see the following screen.

<div align="center">
    <img src="images/geneious-3.png" width="600" alt="Quality Scores Line Plot">
</div> 


**2. Install Geneious**

Ideally, install the newest version of Geneious. Go to [Downloads](https://www.geneious.com/updates) and install the version of Geneious suitable for your operating system. The license will work with previous version of Geneious, but you may miss some of the new features.

<div align="center">
    <img src="images/geneious-4.png" width="600" alt="Quality Scores Line Plot">
</div> 

**3. Get license**

Send and e-mail to Juan Jovel to get an invitation as license user. Please indicate the email to which you want each license to be associated with (ucalgary email addresses are preferred). Each license is valid for two computers.

**4. Activate your license in the Geneious app**

Open the Geneious App. At some point you will see an box like the following:

<div align="center">
    <img src="images/geneious-6.png" width="400" alt="Quality Scores Line Plot">
</div> 

Just click on `activate license`, You will see the following dialog box:

<div align="center">
    <img src="images/geneious-7.png" width="300" alt="Quality Scores Line Plot">
</div> 

- Select `Email Sign in`.
- Provide your email and password (the same ones you use to sign in in the geneious webpage).

If succesful, you should see the following dialog box:

<div align="center">
    <img src="images/geneious-8.png" width="300" alt="Quality Scores Line Plot">
</div> 

That's it. You are set, go and crack sequences, genomes, clones, etc.
